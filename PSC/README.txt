Auxiliary material for Paper #
Retrieval of phytoplankton communities and chlorophyll a concentration from in situ fluorescence profiles using neural networks

R. Sauzede, Laboratoire d'Océanographie de Villefranche, UMR 7093 CNRS, Université Pierre et Marie Curie-Paris 6, Quai de la Darse BP 08, 06238 Villefranche-Sur-Mer, France 

PSC

This sub-directory contains the functions to retrieve simultaneously the chlorophyll-a concentrations associated with three phytoplankton size classes (microphytoplankton,nanophytoplankton and picophytoplankton) for ten normalized depths for each fluorescence profile.
The R script is FLAVOR_PSC.R and the example is done with the validation dataset of the paper which is contained in the file validation_dataset_psc.dat
